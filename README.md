# Local usage

-   `git clone https://github.com/h4x0rlol/avito-test-task`
-   `cd avito-test-task`
-   `yarn`
-   `yarn start`

# Local development

-   `git clone https://github.com/h4x0rlol/avito-test-task`
-   `cd avito-test-task`
-   `yarn`
-   `yarn husky install`
-   `yarn start`
-   run tests with `yarn test`
